## 3. Phase 1：架构清理（已落地）

参考 [evolution-roadmap.md Phase 1](../../docs/architecture/evolution-roadmap.md#phase-1-架构清理已部分落地持续收敛)。

**关键交付**：
- `bootstrap/` 统一入口（app_factory / routes / middleware / lifespan）。
- Agent runtime 重构：Application / Advisor / Runtime / Planner / Executor / Reflector / Response / Sessions。
- Prompt / Context / Memory 工程边界落地。
- `core/` 拆分为 `core/config` / `core/logging` / `core/security`。
- 死代码删除。

**仍在收尾**：
- `services/` 模块化（Crop / Cycle / Ledger / Weather 等）。
- 兼容入口清理。

## 4. Phase 2：多用户 + 数据安全（已落地）

参考 [evolution-roadmap.md Phase 2](../../docs/architecture/evolution-roadmap.md#phase-2-多用户--数据安全主体已落地)。

**关键交付**：
- JWT 认证（access 1h / refresh 30d）。
- 生产数据库从 SQLite 迁移到 MySQL 8.x（utf8mb4）。
- 20 张业务表迁移校验通过。
- Alembic 成为唯一迁移工具。
- Farm 隔离 + 越权测试覆盖。
- 反馈收集 API。
- Token 配额基础。

## 5. Phase 3：功能增强（进行中）

参考 [evolution-roadmap.md Phase 3](../../docs/architecture/evolution-roadmap.md#phase-3-功能增强主体已落地继续增强)。

**关键交付**：
- 双天气 Provider（QWeather + Open-Meteo）+ 预警注入。
- 会话管理（多轮、长程、归档）。
- Context 工程化：Selector / TokenBudget / Compressor / Cache / Preload。
- Memory 工程化：MemoryService + observation。
- Smart Fill（移动端统一入口 `/api/v1/scenarios`）。
- Skill 扩展至 30+。

**剩余任务**：
- runtime 收敛对 Context 的直接构建。
- 长期 Memory 检索（占位）。
- 评测闭环。

## 6. Phase 4：智能化（待启动）

参考 [evolution-roadmap.md Phase 4](../../docs/architecture/evolution-roadmap.md#phase-4-智能化-1-2月)。

**目标**：从"功能完成"到"质量稳定"。

**关键交付**：
- **Reflector 完整化**：幻觉检测、工具结果一致性、Prompt 泄露检测。
- **DataFlywheel 闭环**：
  - 样本治理（采集 / 标签 / 隐私脱敏）
  - 规则候选 + LLM 预标注
  - 人工确认工作流
  - 回归测试集回流
  - 评测 / SFT 数据出口
- **Memory 会话摘要**：running summary 自动生成、持久化与上下文注入已进入 Phase A。
- **Memory 长期**：偏好 / 事件 / 习惯 / 别名 / 事实 5 类。
- **Simulation 完整化**：smoke / regression / nightly / release 四档。
- **Evaluation 完整化**：核心 / 近期 / 对抗三集 + LLM judge。

**成功标准**：
- `task_completion_rate` ≥ 0.90
- `hallucination_rate` ≤ 0.05
- DataFlywheel 月均产出 50+ 高质量样本

## 7. Phase 5：规模化（待启动）

参考 [evolution-roadmap.md Phase 5](../../docs/architecture/evolution-roadmap.md#phase-5-规模化-2-3月)。

**目标**：从"10 农户"到"200+ 农户"。

**关键交付**：
- 性能优化（DB 索引、连接池、缓存、Prompt 缓存）。
- 监控告警（Grafana + Alertmanager）。
- 灰度发布（Feature Flag + A/B）。
- 压测基线 + 容量规划。
- 备份恢复演练（每季度）。
- Admin Web 全面增强（飞轮 / Trace / Skill 管理 / Prompt 管理 / Token 监控）。

**成功标准**：
- 200+ 农户稳定运行
- P95 延迟 < 8s
- 单户月成本 < ¥20

## 8. Phase 6：商业化（待启动）

- 计费（按农场 / 按 token / 按功能）。
- 订阅 / 试用 / 退款。
- SLA 承诺。
- 多区域部署（如腾讯云 + 阿里云双活）。
- 渠道扩展（小程序 / Web）。
