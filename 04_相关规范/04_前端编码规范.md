## 2. 文件组织

### 2.1 Admin Web

```
admin-web/src/
├── api/              # API 客户端（不依赖 components/layouts/pages）
├── components/       # 通用组件
├── constants/        # 常量
├── layouts/          # 布局（不依赖 api/components/pages）
├── pages/            # 页面（可依赖 api/components）
├── stores/           # Zustand stores
├── styles/           # 全局样式
├── utils/            # 工具
├── App.tsx
└── main.tsx
```

依赖方向（强制）：`api → components → layouts → pages`，反向禁止。

### 2.2 Mobile App

```
mobile-app/lib/
├── app/              # App 入口 + 路由
├── data/             # 数据层（api client / model / repo）
├── features/         # 业务特性
├── shared/           # 共享 widgets / assets
├── theme/            # 主题
└── main.dart
```

依赖方向：`shared → data → features → app`，反向禁止。

## 3. 命名规范

### 3.1 文件名

| 类型 | 规则 | 示例 |
| --- | --- | --- |
| React 组件 | PascalCase.tsx | `DataFlywheelTable.tsx` |
| 工具函数 | camelCase.ts | `formatCost.ts` |
| 类型定义 | PascalCase.types.ts 或同文件 | `Trace.ts` |
| 常量文件 | camelCase.constants.ts | `skillTriggers.constants.ts` |
| Dart 文件 | snake_case.dart | `home_screen.dart` |
| Dart 类 | PascalCase | `class HomeScreen` |

### 3.2 标识符

| 类型 | 规则 | 示例 |
| --- | --- | --- |
| 组件 | PascalCase | `function DataFlywheelTable()` |
| 函数 | camelCase | `const formatCost = () => ...` |
| 变量 | camelCase | `const farmId = 1` |
| 常量 | UPPER_SNAKE | `const MAX_RETRY = 3` |
| 类型 / Interface | PascalCase | `interface CostRecord` |
| Enum | PascalCase + PascalCase 成员 | `enum Status { Pending, Done }` |

## 4. TypeScript 规范

### 4.1 类型优先

```typescript
// ✅
interface CostRecord {
  id: number;
  amount: number;
  category: string;
}

// ❌
const record: any = ...
```

### 4.2 异步

```typescript
// ✅ async/await
async function loadCosts() {
  const res = await api.get('/costs');
  return res.data;
}

// ❌ .then 链
function loadCosts() {
  return api.get('/costs').then(res => res.data);
}
```

### 4.3 错误边界

```tsx
<ErrorBoundary fallback={<ErrorView />}>
  <DataFlywheelTable />
</ErrorBoundary>
```

API 异常统一在 `api/client.ts` 拦截，业务层不重复处理。

## 5. 组件规范

### 5.1 函数组件 + Hooks

```tsx
function DataFlywheelTable() {
  const { data, isLoading } = useQuery({
    queryKey: ['data-flywheel'],
    queryFn: () => api.get('/admin/data-flywheel/samples'),
  });

  if (isLoading) return <Spin />;
  return <Table dataSource={data} columns={columns} />;
}
```

### 5.2 拆分原则

| 信号 | 拆分 |
| --- | --- |
| 组件超 150 行 | 拆 sub-components |
| 重复 UI 模式 | 抽通用组件 |
| 跨页面共享 | 放 `components/` |
| 单页特有 | 留在 `pages/<Page>/components/` |

### 5.3 Props 接口

```tsx
interface CostListProps {
  farmId: number;
  pageSize?: number;
  onRowClick?: (record: CostRecord) => void;
}

function CostList({ farmId, pageSize = 20, onRowClick }: CostListProps) { ... }
```

**禁止**：Props 用 `any`。

## 6. 状态管理

### 6.1 局部状态

`useState` / `useReducer`。

### 6.2 跨组件共享（Zustand）

```ts
// stores/authStore.ts
interface AuthState {
  user: User | null;
  login: (username, password) => Promise<void>;
  logout: () => void;
}

export const useAuthStore = create<AuthState>((set) => ({
  user: null,
  login: async (username, password) => {
    const res = await authApi.login(username, password);
    set({ user: res.user });
  },
  logout: () => set({ user: null }),
}));
```

### 6.3 服务端状态（React Query）

```ts
const { data, isLoading } = useQuery({
  queryKey: ['costs', farmId],
  queryFn: () => costsApi.list(farmId),
  staleTime: 60_000,
});
```

## 7. API 客户端

```ts
// api/client.ts
const client = axios.create({
  baseURL: '/api/v1',
  timeout: 30_000,
});

client.interceptors.request.use((config) => {
  const token = useAuthStore.getState().token;
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

client.interceptors.response.use(
  (res) => res,
  (err) => {
    if (err.response?.status === 401) {
      useAuthStore.getState().logout();
      window.location.href = '/login';
    }
    return Promise.reject(err);
  }
);
```

## 8. Flutter 规范

### 8.1 Widget 风格

```dart
class FarmDashboard extends ConsumerWidget {
  const FarmDashboard({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final farmAsync = ref.watch(currentFarmProvider);
    return farmAsync.when(
      data: (farm) => _buildContent(farm),
      loading: () => const LoadingView(),
      error: (e, _) => ErrorView(message: e.toString()),
    );
  }
}
```

### 8.2 设计系统（强制）

详见 [.claude/rules/app-ui.md](../../.claude/rules/app-ui.md)。核心：

## 10. 性能规范

| 场景 | 策略 |
| --- | --- |
| 列表 | 虚拟滚动（AntD Table / ListView.builder） |
| 图片 | 懒加载 + 占位符 |
| 重渲染 | React.memo / useMemo / useCallback |
| Flutter 重建 | Selector / Provider.select 精准订阅 |
| 包体积 | Tree shaking + 动态 import |

## 11. 可访问性（预留）

- 触摸目标 ≥ 44×44（移动）
- 颜色对比 ≥ 4.5:1
- 表单 label 完整
- 键盘可达（Admin Web）
