# 02 — 非功能性需求

## 1. 性能

| 指标 | 2026 目标 | 2027 目标 |
| --- | --- | --- |
| 首字节延迟（SSE 第一个 token） | p50 ≤ 1.5s / p95 ≤ 2.5s | p50 ≤ 1s / p95 ≤ 1.5s |
| 完整回复延迟（一次性） | p95 ≤ 5s | p95 ≤ 3s |
| 工具调用延迟（单 Skill） | p95 ≤ 1s | p95 ≤ 500ms |
| 每日建议生成 | 单农场 ≤ 10s | ≤ 5s |
| 列表页 API | p95 ≤ 200ms | ≤ 100ms |
| 详情页 API | p95 ≤ 500ms | ≤ 300ms |
| 并发用户 | 50 | 500 |
| 并发 LLM 请求 | 20 | 100 |

## 2. 可用性

| 指标 | 目标 |
| --- | --- |
| 月可用率 | ≥ 99.5%（2C4G 部署，不强求 99.9%） |
| 计划内停机 | ≤ 2 小时/月（凌晨维护） |
| 单点故障恢复 RTO | ≤ 1 小时 |
| 单点故障恢复 RPO | ≤ 1 小时（MySQL binlog） |
| LLM Provider 故障降级 | ≤ 30s 切到国产 Provider |

## 3. 成本

| 指标 | 2026 目标 | 2027 目标 |
| --- | --- | --- |
| 月 LLM 成本（10 用户） | ≤ 100 USD | ≤ 500 USD（200 用户） |
| 单用户月成本 | ≤ 10 USD | ≤ 2.5 USD |
| Token 节省（Prompt Caching） | ≥ 60% | ≥ 75% |
| 服务器成本（月） | ≤ 200 RMB | ≤ 800 RMB |
| 单用户月毛利 | ≥ 30 RMB | ≥ 50 RMB |

成本超阈值告警见 [07_可观测与运维](../01_正式设计/07_可观测与运维.md)。

## 4. 安全

详见 [04_相关规范/02_安全与权限规范](../04_相关规范/02_安全与权限规范.md)。核心：

- JWT 短期 + Refresh Token
- 密码 bcrypt hash
- 所有 API 入参 Pydantic 校验
- SQL 注入防护：只用 ORM
- 越权防护：Farm 隔离（current_farm Depends）
- 敏感字段序列化时排除（password、token、API key）
- 输入/输出 Guardrails（敏感词、超长、注入检测）
- Pending Action 二次确认所有写操作

## 5. 可维护性

| 指标 | 目标 |
| --- | --- |
| 单文件行数 | ≤ 500 |
| 单方法行数 | ≤ 50 |
| 单类行数 | ≤ 200 |
| 方法参数 | ≤ 5（超过用 dataclass） |
| 单行字符 | ≤ 120 |
| 测试覆盖率 | ≥ 80% |
| 关键路径覆盖率 | ≥ 95% |
| Lint 通过率 | 100%（CI 阻塞） |
| 架构依赖违规 | 0（CI 阻塞） |

## 6. 可扩展性

- 横向扩展：2C4G 单机够用至 500 用户；超 500 考虑拆 LLM Worker
- 数据扩展：MySQL 主备 + JSONL 异步备份；可平滑迁到云数据库
- LLM Provider：多 Provider 路由，新增 Provider 实现端口即可
- 天气 Provider：双源策略，新增只需实现 WeatherProvider Protocol
- Skill 扩展：新增 Skill 按契约实现，自动注册

## 8. 国际化

当前优先中文（简体）：
- 提示词、UI 文案、错误消息中文优先
- 时间格式 `YYYY-MM-DD`
- 时区：东八区
- 货币：人民币（分）
- 单位：亩、公斤、毫米（农业习惯）

英文 i18n 留到 2027。

## 10. 可观测性

详见 [01_正式设计/07_可观测与运维](../01_正式设计/07_可观测与运维.md)。
